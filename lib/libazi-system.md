# 李八子&号神 实体IP爆款体系（参考知识库）

## 一、三大固定镜头（每条视频必混剪）
1. **正面近景口播**（信任镜头）：胸口以上、平视、自然光、不摆姿势
2. **行走纪实镜头**（真实感核心）：手持轻微晃动、边走边说
3. **细节特写镜头**（价值镜头）：产品材质、工艺、手部动作

## 二、剪辑固定法则
- 时长：23±2s
- 结构：0-3s钩子 → 3-19s正文 → 19s-结尾引流收口
- 字幕：黑体加粗居中短句，3s钩子放大1.5倍，无花字贴纸
- BGM：轻柔钢琴纯音乐，音量≤人声20%
- 剪辑：只删卡顿口头禅，保留自然停顿，不加速人声
- 禁止：无美颜磨皮、无摆拍剧情、无提词器眼神漂移

## 三、文案四核心
痛点抓人、行业真话、真诚人设、私信收口
